# Provenance Enforcement Design

## Problem Statement

Memory artifacts in Duro are currently self-attested. Any caller can claim any `workflow` or `provenance` value. This enables **memory poisoning attacks** where:

1. Attacker stores fact with `workflow="orchestrator"` to appear authoritative
2. Prompt injection triggers auto-capture of malicious "facts"
3. Forged `validated` status on decisions bypasses review

**Goal**: Ensure "the artifact says it's approved" ≠ "it's actually approved"

---

## Design: Trust Tiers + Signed Provenance

### 1. Trust Tier Enum

```python
class TrustTier(IntEnum):
    """
    Trust tiers for artifact provenance.
    Higher numbers = less trust.
    Immutable after creation.
    """
    HUMAN_VERIFIED = 1   # Explicit user input, human-validated decisions
    SYSTEM_INTERNAL = 2  # Generated by Duro internals (orchestrator, policy gate)
    AUTO_CAPTURED = 3    # Extracted from conversations, auto-generated
    EXTERNAL = 4         # From web, files, tool output, unknown source
```

**Rules**:
- Tier is **computed at creation time** based on provenance, not claimed
- Tier is **immutable** - cannot be upgraded without human validation
- Lower tier (more trusted) can reference higher tier, not vice versa

### 2. Workflow Allow-List

Only internal Duro code can claim certain workflows. External MCP calls get downgraded.

```python
# In artifacts.py

# Workflows that can only be set by internal code paths
INTERNAL_WORKFLOWS: frozenset = frozenset({
    "orchestrator",
    "policy_decision",
    "security_audit",
    "debug_rca",
    "change_ledger",
    "promotion",
})

# Mapping: claimed workflow -> trust tier
WORKFLOW_TRUST_MAP: dict = {
    # Tier 1: Human-verified
    "manual": TrustTier.HUMAN_VERIFIED,
    "user_input": TrustTier.HUMAN_VERIFIED,

    # Tier 2: System internal
    "orchestrator": TrustTier.SYSTEM_INTERNAL,
    "policy_decision": TrustTier.SYSTEM_INTERNAL,
    "debug_rca": TrustTier.SYSTEM_INTERNAL,
    "change_ledger": TrustTier.SYSTEM_INTERNAL,

    # Tier 3: Auto-captured
    "auto_capture": TrustTier.AUTO_CAPTURED,
    "session": TrustTier.AUTO_CAPTURED,
    "extraction": TrustTier.AUTO_CAPTURED,

    # Tier 4: External
    "web_fetch": TrustTier.EXTERNAL,
    "tool_output": TrustTier.EXTERNAL,
    "unknown": TrustTier.EXTERNAL,
}

def compute_trust_tier(
    workflow: str,
    provenance: str | None = None,
    is_internal_caller: bool = False
) -> TrustTier:
    """
    Compute trust tier based on provenance metadata.

    SECURITY: External callers cannot claim internal workflows.
    If they try, tier is downgraded to EXTERNAL.
    """
    # If claiming internal workflow but not internal caller, downgrade
    if workflow in INTERNAL_WORKFLOWS and not is_internal_caller:
        return TrustTier.EXTERNAL

    # Use workflow mapping
    tier = WORKFLOW_TRUST_MAP.get(workflow, TrustTier.EXTERNAL)

    # Provenance can only downgrade, never upgrade
    if provenance in ("web", "tool_output", "unknown"):
        tier = max(tier, TrustTier.EXTERNAL)

    return tier
```

### 3. Provenance Signature (HMAC)

Sign artifacts at creation to detect tampering.

```python
import hmac
import hashlib
from typing import NamedTuple

class ProvenanceSignature(NamedTuple):
    """Cryptographic binding of provenance to artifact."""
    artifact_id: str
    trust_tier: int
    workflow: str
    created_at: str
    signature: str  # HMAC-SHA256

def sign_provenance(
    artifact_id: str,
    trust_tier: TrustTier,
    workflow: str,
    created_at: str,
    secret: bytes
) -> ProvenanceSignature:
    """
    Create HMAC signature binding provenance to artifact.

    The signature proves:
    1. This artifact was created by Duro (has the secret)
    2. The trust tier was computed at creation time
    3. Provenance hasn't been tampered with
    """
    message = f"{artifact_id}:{trust_tier.value}:{workflow}:{created_at}"
    sig = hmac.new(secret, message.encode(), hashlib.sha256).hexdigest()

    return ProvenanceSignature(
        artifact_id=artifact_id,
        trust_tier=trust_tier.value,
        workflow=workflow,
        created_at=created_at,
        signature=sig
    )

def verify_provenance(
    sig: ProvenanceSignature,
    secret: bytes
) -> bool:
    """Verify provenance signature hasn't been tampered with."""
    message = f"{sig.artifact_id}:{sig.trust_tier}:{sig.workflow}:{sig.created_at}"
    expected = hmac.new(secret, message.encode(), hashlib.sha256).hexdigest()
    return hmac.compare_digest(sig.signature, expected)
```

### 7. Upgrade Path: Human Validation

Trust tier can only be upgraded through explicit human validation:

```python
def validate_and_upgrade_trust(
    artifact_id: str,
    new_tier: TrustTier,
    validator: str,
    reason: str,
    secret: bytes
) -> bool:
    """
    Upgrade artifact trust tier through human validation.

    RULES:
    - Can only upgrade (lower tier number), never downgrade
    - Requires explicit human action (validator identity)
    - Creates audit trail
    - Re-signs with new provenance
    """
    artifact = get_artifact(artifact_id)
    current_tier = artifact["provenance"]["trust_tier"]

    # Can only upgrade
    if new_tier >= current_tier:
        return False

    # Record validation
    artifact["provenance"]["validators"].append({
        "validator": validator,
        "from_tier": current_tier,
        "to_tier": new_tier,
        "reason": reason,
        "timestamp": utc_now_iso()
    })

    # Update tier and re-sign
    artifact["provenance"]["trust_tier"] = new_tier
    artifact["provenance"]["signature"] = sign_provenance(
        artifact_id, new_tier, artifact["provenance"]["workflow"],
        artifact["created_at"], secret
    ).signature

    return True
```

## Security Properties

| Property | Mechanism |
|----------|-----------|
| **Forgery resistance** | HMAC signature with server secret |
| **Tier immutability** | Computed at creation, can only upgrade via validation |
| **Workflow verification** | Allow-list prevents external claiming of internal workflows |
| **Tamper detection** | Signature verification on retrieval |
| **Audit trail** | Validators array tracks all trust changes |

---

## Attack Mitigation

| Attack | Before | After |
|--------|--------|-------|
| Claim `workflow="orchestrator"` | Accepted | Downgraded to EXTERNAL |
| Auto-capture poisoning | Surfaced equally | Marked as tier 3, filterable |
| Forge `validated` status | No detection | Signature mismatch |
| Modify stored provenance | Undetected | Signature verification fails |
| Escalate trust tier | Set any value | Can only upgrade via human validation |
