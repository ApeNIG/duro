- ✅ Decision validation lifecycle
- ✅ 48-hour debug rule
- ✅ Orchestration and autonomy
- ✅ Audit logging

## Philosophy

Duro prioritizes:

1. **Local-first**: Your data stays on your machine
2. **Simple**: JSON files, SQLite, no cloud required
3. **Composable**: Works with any MCP client
4. **Transparent**: Full audit trail, no magic
