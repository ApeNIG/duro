### [04:51] Learnings
**Learning (Technical):** Hershey fonts are public domain single-stroke vector glyphs (designed for pen plotters) that enable generative typography. Unlike filled fonts, each letter is an SVG path you trace with a single line. The key to organic hand-drawn effects is applying perpendicular wave motion to each stroke using deterministic seeded randomness (Math.sin(seed * 12.9898 + 78.233) * 43758.5453).

### [04:51] Learnings

Each layer has different strokeWidth, opacity, and offset values. The OrganicTextRefined component in InterSphere demonstrates this pattern.

### [10:07] Learnings
**Learning (Technical):** Pencil MCP supports NATIVE GRADIENT FILLS - no need to stack ellipses! Use fill: {type: "gradient", gradientType: "radial"|"linear"|"angular", center: {x, y}, size: {width, height}, colors: [{color, position}...]}. Center uses normalized 0-1 coordinates (0.5, 0.5 = center). For edge glow: set center at the edge (y:1 for bottom, y:0 for top). This gives true mathematical interpolation - perfectly smooth, no banding, fewer nodes.
