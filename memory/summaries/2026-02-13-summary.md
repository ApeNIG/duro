## Learnings
- Hershey fonts are public domain single-stroke vector glyphs (designed for pen plotters) that enable generative typography. Unlike filled fonts, each letter is an SVG path you trace with a single line. The key to organic hand-drawn effects is applying perpendicular wave motion to each stroke using deterministic seeded randomness (Math.sin(seed * 12.9898 + 78.233) * 43758.5453).
- Production-quality generative typography uses 4 layers: (1) blur layer for soft glow, (2) shadow layer for depth, (3) core layer for main strokes, (4) accent layer for highlights. Each layer has different strokeWidth, opacity, and offset values. The OrganicTextRefined component in InterSphere demonstrates this pattern.
- Internal design documentation pages (like /demo/lines) that serve as both live demos AND technical references are extremely valuable. They preserve design iteration history, provide API documentation, show roadmap status, and let designers/developers see all variations in one place. Worth creating for any significant visual system.

- When creating designs in Pencil MCP, parent frames need `layout: "vertical"` set explicitly, otherwise child elements get positioned off-screen (at x positions equal to the frame width). Always set layout property on container frames.
- For glassmorphism effects in Pencil: use semi-transparent fill colors (e.g., #27272780) so background elements show through. Pencil doesn't support blur filters, but transparency creates the effect.
- When recreating designs, always self-critique against design principles (hierarchy, contrast, balance, alignment, proximity) before presenting. A design can look "similar" but fail fundamentally if core principles are violated.
- PDF/image to mockup workflow: 1) Extract content (text, structure, data), 2) Identify the visual style, 3) Recreate with different treatment while preserving information hierarchy. User can specify match percentage (50%, 90%) to control how much original style to retain.
- Glassmorphism requires transparency + visible background elements to "read" as glass. Use semi-transparent fills (#RRGGBBAA format) with 50-70% opacity, and ensure background shapes are visible through the card.
